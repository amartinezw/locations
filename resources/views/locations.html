Locations

API Endpoint
https://locationsback.ccp.tienda/api/v1

La interfaz para programación de la aplicación (API) Locations está desarrollada en Lumen 5.8 y está basada en arquitectura REST que utiliza el protocolo HTTP para proveer una interfaz uniforme, agnóstica a lenguajes de programación, orientado a recursos y con códigos de respuesta estandarizados. Todas las respuestas del API están en formato JSON y utiliza los códigos de respuesta del protocolo HTTP en los encabezados.
